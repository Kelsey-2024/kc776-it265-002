#### Progress
- **What I accomplished**:
  - Finalized the title of the game: *Into the Shadows*.

  - Designed the main gameplay mechanics including tile exploration, event cards, trap encounters, item collection, and cooperative gameplay.

- **Challenges faced**:
  - One challenge was determining how the board should function since the game does not start with a fixed map. I needed to figure out how room tiles would be placed and how exploration would reveal new areas without confusing players.
  - Another challenge was balancing randomness with strategy. Because tiles and events drawn during gameplay, I had to think about how to keep the game fair while still maintaining unpredictability.
  - Designing the card system was also difficult because I needed to decide how many types of cards should exist and what effects they should trigger.

- **Solutions**:
  - I addressed the map exploration challenge by implementing a modular tile system where new rooms are drawn from a stack and placed adjacent to the current room when players explore.
  - To balance randomness and strategy, I designed different categories of cards such as helpful items, dangerous traps, and narrative events so that players experience both positive and negative outcomes.
  - I simplified the game flow by creating a clear turn structure: move, reveal tile, resolve event, and end turn.
